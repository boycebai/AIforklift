文件所需配置环境如下：

- python    `2.7.12`

- cv2       `3.4.2`

- numpy     `1.15.4`

- imtils    `0.5.1`

- tqdm      `4.19.4`

- moviepy   `0.2.3.5`



opencv-python 3.4.2.16

opencv-contrib-python 3.4.2.16



"video_stitcher_better.py"进行了视频合成处理，并进行了缝隙修复的后处理，运行速度较慢
执行命令：python video_stitcher_better.py


"video_stitcher_fast.py"进行了视频合成处理，未进行缝隙修复的后处理，运行可以达到实时
执行命令：python video_stitcher_fast.py
