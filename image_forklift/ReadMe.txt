文件所需配置环境如下：

- python    `2.7.12`

- cv2       `3.4.2`

- numpy     `1.15.4`

- Pillow    `3.1.2`



WINDOWS下操作流程：
1)打开POWERSHELL

2)定位到代码所在位置:
`\image_forklift\code

3)执行程序demo：
python .\navigation_main_1030.py -I
修改文件内注释，即可读单张图像也可读视频

或者接上深度摄像头,执行程序:
python .\navigation_main_1030.py


4）按住回车键ENTER，可持续运行
